# Installation

In your terminal navigate to the folder you want to clone this repo into and clone with 
```
git clone https://github.com/Landscape-CV/PyTLiDAR.git
cd PyTLiDAR
python -m venv .venv
```
### Mac
```
. .venv/bin/activate
```
### Windows
```
. .venv/Scripts/activate
```
#
```
pip install -r requirements.txt
```
